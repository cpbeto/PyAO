### Instrucciones p/ Linux (Ubuntu 24.04)

1. Crear un entorno virtual de Python e instalar el motor Ursina.
```bash
python3 -m venv ursinaenv
source ursinaenv/bin/activate
pip install ursina
```

2. Ejecutar haciendo `python3 PyAO/main.py`
